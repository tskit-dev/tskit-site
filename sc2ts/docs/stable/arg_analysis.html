ARG analysis#

The sc2ts API provides some convenience functions to compute summary dataframes for the nodes and mutations in a sc2ts-output ARG.

Prerequisites#

Download a subset of the sc2ts Viridian ARG with 1000 samples:

curl -O https://raw.githubusercontent.com/tskit-dev/sc2ts/refs/heads/main/docs/sc2ts_viridian_v1.2_subset_1000.trees.tsz

We’ll use this small subset as an example throughout.

Loading#

import sc2ts
import tszip

ts = tszip.load("sc2ts_viridian_v1.2_subset_1000.trees.tsz")

You can then use the full tskit Python API on this ARG.

Node data#

The node_data() function returns a Pandas dataframe of data for each node in the ARG.

dfn = sc2ts.node_data(ts)
