Python API#

This page documents the public Python API exposed by sc2ts. Inference is driven via the command line interface (see the CLI documentation); the functions and classes listed here are intended for working with tree sequences and datasets that have already been generated.

ARG analysis#

node_data(ts[, inheritance_stats])

Return a dataframe of per-node statistics for a sc2ts tree sequence.

mutation_data(ts[, inheritance_stats, ...])

Return a dataframe of per-mutation statistics for a sc2ts tree sequence.
sc2ts.node_data(ts, inheritance_stats=False)[source]#

Return a dataframe of per-node statistics for a sc2ts tree sequence.

The input ts must be the output of the sc2ts minimise-metadata CLI command (debug metadata from raw inference runs is not supported). Each row in the returned pandas.DataFrame corresponds to a node ID and includes basic metadata, flags and mutation counts, with optional inheritance statistics.

Warning

Computing inheritance_stats requires numba, which is not installed in the minimal version of sc2ts for portability reasons. You will need to install the full “inference” build of sc2ts, or install numba manually to access this feature.

Parameters:

  • ts (tskit.TreeSequence) – Tree sequence produced by the sc2ts minimise-metadata CLI command.

  • inheritance_stats (bool) – If True, include additional statistics summarising the inheritance patterns of each node. Defaults to False.

Returns:

A dataframe of per-node statistics

Return type:

pandas.DataFrame

sc2ts.mutation_data(ts, inheritance_stats=False, parsimony_stats=False)[source]#

Return a dataframe of per-mutation statistics for a sc2ts tree sequence.

The input ts must be the output of the sc2ts minimise-metadata CLI command. Each row in the returned pandas.DataFrame corresponds to a mutation ID and includes positional information, states, and optional inheritance and parsimony statistics.

Warning

Computing inheritance_stats requires numba, which is not installed in the minimal version of sc2ts for portability reasons. You will need to install the full “inference” build of sc2ts, or install numba manually to access this feature.

Parameters:

  • ts (tskit.TreeSequence) – Tree sequence produced by the sc2ts minimise-metadata CLI command.

  • inheritance_stats (bool) – If True, include descendant and inheritor counts for each mutation. Defaults to False.

  • parsimony_stats (bool) – If True, compute additional parsimony-based diagnostics such as flagging immediate reversions. Defaults to False.

Returns:

Dataframe of per-mutation statistics

Return type:

pandas.DataFrame

class sc2ts.Dataset(path, chunk_cache_size=1, date_field=None)[source]#

Open a sc2ts VCF Zarr dataset for convenient access to alignments and metadata.

The dataset is opened read-only from path, which may be either a directory store or a consolidated .zip file.

Parameters:

  • path (str) – Path to a directory or .zip Zarr store.

  • chunk_cache_size (int) – Maximum number of chunks to cache for alignments and metadata. Defaults to 1.

property alignment#

Efficient mapping of sample ID strings to integer encoded alignment data.

The returned object is dictionary-like implemening the Mapping protocol. Access to the underlying Zarr store is mediated by a chunk cache, so that chunks are not repeatedly decompressed.

property metadata#

Efficient mapping of sample ID strings to metadata dictionaries.

The returned object is dictionary-like implemening the Mapping protocol. Access to the underlying Zarr store is mediated by a chunk cache, so that chunks are not repeatedly decompressed.

metadata_dataframe(fields=None)[source]#

Returns the metadata in this dataset as a Pandas dataframe, indexed by sample_id.

Parameters:

fields – List of metadata fields to include; if None, all fields are used.

Pandas dataframe

property sample_id#

The sample IDs as a numpy string array.

property num_samples#

Return the number of samples in this dataset.

property num_variants#

Return the number of variants in this dataset. Note that this does not mean that there’s necessarily variation at each site; the terminology is borrowed from VCF Zarr.

variants(sample_id=None, position=None)[source]#

Iterate over variants at the specified positions for the given samples.

Yields Variant objects containing the genomic position, encoded genotypes, and allele labels for each requested site.

Parameters:

  • sample_id – Iterable of sample IDs to include; if None, all samples are used.

  • position – Iterable of genomic positions to extract; if None, all variant positions are used.

Returns:

Generator over Variant records for the requested sites.

collections.abc.Iterator

sc2ts.decode_alleles(a)[source]#

Decode an array of integer-encoded alleles into their IUPAC string values returned as a numpy array.

The input should use the encoding defined by core.IUPAC_ALLELES, with -1 representing missing data.

Parameters:

a (numpy.ndarray) – Integer-encoded alignment array.

Returns:

Array of single-character IUPAC allele codes.

Return type:

numpy.ndarray

sc2ts.mask_ambiguous(a)[source]#

Mask all ambiguous and non-standard alleles in an encoded alignment array.

Values corresponding to ambiguous IUPAC codes (and other values beyond the deletion code) are replaced with -1 in a copy of a, leaving the original array unchanged.

Parameters:

a (numpy.ndarray) – Integer-encoded alignment array.

Returns:

Copy of a with ambiguous alleles replaced by -1.

Return type:

numpy.ndarray

sc2ts.mask_flanking_deletions(a)[source]#

Mask sites flanking the first and last non-deletion allele in an alignment.

Positions before the first non-deletion and after the last non-deletion in the encoded alignment are set to -1 in a copy of a. If the alignment consists entirely of deletions, all sites are masked.

Parameters:

a (numpy.ndarray) – Integer-encoded alignment array.

Returns:

Copy of a with flanking sites masked to -1.

Return type:

numpy.ndarray

sc2ts.decode_flags(f)[source]#

Return the list of FlagValue records set in the given flag mask.

The input f is interpreted as a bitmask of tskit and sc2ts-specific node flags. Each returned FlagValue describes a single flag, including its integer value, short code, long name, and text description.

Parameters:

f (int) – Integer bitmask of node flags to decode.

Returns:

A list of flag descriptors corresponding to the bits set in f.

Return type:

list

sc2ts.flags_summary(f)[source]#

Return a compact string summarising the flags set in the given mask.

Each character in the returned string corresponds to a known flag; set flags are shown using their short code and unset flags as "_".

Parameters:

f (int) – Integer bitmask of node flags to summarise.

Returns:

Summary string showing short codes for set flags.

Return type:

str
