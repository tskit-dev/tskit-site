The tskit ecosystem for your research

The tskit ecosystem can be used for a wide range of genetic research purposes. The tree sequence file format is not only an efficient, scalable way to store and analyse genotype data, but as its underlying structure is a set of one or more trees it can be used to store and analyse phylogenetic data.

Below we summarise some of the key features of the tskit ecosystem for use in four key areas:

Population genetics

The tree sequence file format stores population genetic data in a fraction of the space required by table-based methods. The tskit library itself then provides a number of methods for efficient calculation of population genetic statistics.

Phylogenetics

Along with the tree sequence format that can store phylogenetic trees, tskit also has methods to generate and compare topologies. Efficient access to the trees' structure is provided by the quintuply linked tree arrays.

Use tskit in your application

tskit provides both a Python and C API for reading, writing and manipulating tree sequences. This can be used by simulations and inference tools to write tree sequences to disk via their table representation, and by analysis tools to read them, decode their genotypes or perform statistical measurements using the tree sequence structure for efficiency. Examples of using the tskit C API can be found here and extensive Python API usage is found in the tskit tutorials.
