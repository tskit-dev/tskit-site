Population-scale genomics

tskit for genetics research

Tskit enables performant storage, manipulation and analysis of genomes and phylogenies using succinct tree sequences usable from Python, C and R. Other software programs use tskit, forming an interoperating ecosystem of tools.

This site is a collection of resources to help you learn about tree sequences and associated software, and help you join the online community of open source developers that makes it all possible.
